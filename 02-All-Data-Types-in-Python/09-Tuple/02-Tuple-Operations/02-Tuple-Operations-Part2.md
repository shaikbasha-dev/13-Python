# Concatenation

## Definition

Concatenation means combining two or more tuples into a single tuple.

The `+` operator is used for concatenation.

---

## Program

```python
tuple1 = (10, 20)

tuple2 = (30, 40)

result = tuple1 + tuple2

print(result)
```

---

## Output

```text
(10, 20, 30, 40)
```

---

## Explanation

The `+` operator combines both tuples into one new tuple.

# Repetition

## Definition

Repetition means repeating the elements of a tuple multiple times.

The `*` operator is used for repetition.

---

## Program

```python
numbers = (10, 20)

print(numbers * 3)
```

---

## Output

```text
(10, 20, 10, 20, 10, 20)
```

---

## Explanation

The `*` operator repeats the tuple three times.

# Length Operation

## Definition

The `len()` function returns the total number of elements in a tuple.

---

## Program

```python
languages = ("Python", "Java", "C++")

print(len(languages))
```

---

## Output

```text
3
```

---

## Explanation

The `len()` function counts the total number of elements present in the tuple.
