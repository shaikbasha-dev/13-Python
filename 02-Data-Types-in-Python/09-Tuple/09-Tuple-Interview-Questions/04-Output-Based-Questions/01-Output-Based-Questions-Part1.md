# Python Tuple Output-Based Interview Questions

## Introduction

This document contains output-based interview questions on Python Tuples.

These questions help improve logical thinking and understanding of Python Tuple concepts.

---

# Question 1

## Predict the Output

```python
numbers = (10, 20, 30)

print(numbers)
```

### Output

```text
(10, 20, 30)
```
